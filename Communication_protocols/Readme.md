# 🔌 Communication Protocols – Raspberry Pi Pico Series

This directory contains practical implementations of hardware communication interfaces for:

- Raspberry Pi Pico H
- Raspberry Pi Pico W
- Raspberry Pi Pico 2 W

Supported protocols:

- I2C
- SPI
- UART
- USB Communication

All examples are written in MicroPython and can be configured for any of the above boards
